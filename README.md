# SD E-commerce Website

Un projet de site e-commerce développé avec **HTML, CSS, SQL et JavaScript**.

# Fonctionnalités
- Page d’accueil avec présentation des produits
- Catalogue avec catégories 
- Panier d’achat interactif (Historique => Login)

# Thechnologies utilisés
Bootstrap
Wampserver
